Ошибки:

1) лучше имена методов начинать с большой буквы, но это скорее вопрос конвенции

2) лучше string так не парсить к integer
можно использовать следующую конструкцию:
int i = 0;
Int32.TryParse(b, out i);

3)в условии if() необходимо использовать оператор равенства "==", 
так как он возвращает тип boolean, который как раз прнимает в условие конструкция if()

4) в теле if() возвращается тип boolean, тогда как возвоащаемый тип метода - int

5) не обработаны другие возможные значения переменной result.
Как минимум можно написать после if()
else
{
return result;
}
