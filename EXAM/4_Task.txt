Page Factory
Расширение для page object.
Используется для инициализации веб-элементов для объекта страницы.

Аннотации:
- [FindsBy(How = How.??, Using = "??")] - ищет елемент по данному методу (локатору)
- [CacheLookup] - один раз находим элемент, кешируем его и не ищем постоянно одно и то же.
Экономит время при поиске элементов.
