PAGE OBJECT
Дизайн-паттерн в тестировании.

- каждая web-страница описывается в отдельном классе, которые называются page object

- в каждом page object объявляются элементы, которые находятся на этой странице, а также 
специфические методы, которые относятся только к этой странице

- в отдельные классы выносятся тесты и скрипты

- в отдельные классы выносятся утилиты и ресурсы, необходимые для реализации тестов
